# Feedback Campos obrigatórios

{% hint style="success" %}
Todos os **campos de input** que **requerem o preenchimento** antes de uma **ação do usuário** (ex. acionamento de um botão salvar) devem conter o **“\*”** indicando que o campo é **obrigatório**.
{% endhint %}
